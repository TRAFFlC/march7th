# 三月七虚拟桌宠

基于 Electron 的桌面浮动角色应用，提供便捷的语音对话交互体验。

## 功能特性

- 🎭 **桌面悬浮** - 三月七角色常驻桌面，点击展开聊天
- 💬 **即时对话** - 快速发送消息，获得语音回复
- 🎤 **语音输入** - 支持语音识别转文字
- 🔊 **语音播放** - 自动播放 TTS 合成的语音
- 📍 **系统托盘** - 最小化到托盘，随时唤起
- 🔐 **登录验证** - 需要登录后才能使用

## 交互流程

```
启动桌宠 → 三月七悬浮在桌面角落
              ↓
        点击图标 → 展开聊天面板
              ↓
        输入/语音 → 发送消息
              ↓
        收到回复 → 自动播放语音
              ↓
        关闭面板 → 缩小回托盘
```

## 文件结构

```
desktop_pet/
├── main.js          # Electron 主进程

└── start.bat       # 启动脚本
```

## 快速开始

### 1. 安装依赖

```bash
cd desktop_pet
npm install
```

### 2. 启动后端服务

确保 FastAPI 后端已启动：

```bash
cd ..
python -m uvicorn api.main:app --reload
```

### 3. 启动桌宠

双击运行 `start.bat`，或命令行：

```bash
npm start
```

## 使用方式

### 登录
- 首次使用需要登录
- 使用与主系统相同的账号密码
- 登录信息保存在本地

### 发送消息
- 在输入框输入文字，按 Enter 或点击发送按钮
- 点击麦克风按钮进行语音输入

### 语音交互
- 消息发送后自动播放 TTS 语音回复
- 语音输入使用浏览器 Web Speech API

### 窗口操作
- 点击"−"按钮最小化到托盘
- 点击"×"按钮关闭（实际隐藏到托盘）
- 双击托盘图标切换显示/隐藏

## 系统要求

- Windows 10/11
- Node.js 16+
- 已启动的后端服务 (http://127.0.0.1:8000)

## 注意事项

- 首次运行需要确保后端服务已启动
- 语音输入需要浏览器支持 Web Speech API
- 建议使用 Chrome 或 Edge 浏览器以获得最佳体验
